# 产品图片目录

此目录用于存放所有产品的图片文件。

## 图片命名规则

产品图片应遵循以下命名规则：

1. 主图片：`product-id.jpg`（例如：`perimeter-display.jpg`）
2. 详情图片：`product-id-1.jpg`, `product-id-2.jpg`等

## 图片尺寸建议

为保证网站显示效果一致，建议使用以下图片尺寸：

- **主图片**：1200 x 800 像素
- **详情图片**：1600 x 1000 像素

## 图片格式

支持以下图片格式：
- JPG/JPEG (推荐用于照片类图片)
- PNG (推荐用于需要透明背景的图片)
- WebP (推荐用于优化加载速度)

## 占位图片

目录中的`placeholder`文件夹包含默认占位图片，可在实际产品图片准备好之前临时使用。
